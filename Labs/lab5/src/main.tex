\section{Crafting a Compiler}
\subsection{9.2 -- AST for Signtest Operation}

Assume that we add a new kind of conditional statement to C or Java, the \textit{signtest}. 
Its structure is:
\begin{verbatim}
    signtest ( exp ) {
        neg: stmts
        zero: stmts
        pos: stmts
    }
\end{verbatim}
The integer expression \textit{exp} is evaluated. If it is negative, the statements following neg are executed. If it is zero, the statements following zero are executed. If it is positive, the statements following pos are executed. \\ 


Show the AST you would use for this construct. 

\begin{verbatim}
Signtest
-Compare exp
--PosBlock (if > 0)
---Stmts
--ZeroBlock (if = 0)
---Stmts
--NegBlock (if < 0)
---Stmts 
\end{verbatim}
